## คอร์สเรียนดนตรี
__Data access rights__ `Public` `Follower` `Member` `Private`
- ผู้ใช้โปรไฟล์ประเภท Public
    - สามารถใช้ทางลัดเพื่อเข้าถึง Library/คลัง ได้
    - ส่วนหนึ่งของหน้า Profile/โปรไฟล์ _(เฉพาะการเข้าดูหน้าโปรไฟล์ประเภท Creator/เจ้าของคอร์สเรียน)_
        - สามารถ Subscription/การสมัครสมาชิก ได้
        - สามารถเรียกดูรายละเอียด เงื่อนไข Subscription/การสมัครสมาชิก ได้
    - โดยรวม
        - สามารถเรียกดูรายการคอร์สเรียนทั้งหมดที่ระบบแนะนำให้สำหรับผู้ใช้ โดยที่คอร์สเรียนนั้น ๆ เปิดสิทธิ์การเข้าถึงผู้ใช้โปรไฟล์ประเภท Public ได้
        - สามารถเรียกดูรายการคอร์สเรียนจาก Creator/เจ้าของคอร์สเรียน ที่ได้มีการ Follow ไว้โดยที่คอร์สเรียนนั้น ๆ เปิดสิทธิ์การเข้าถึงผู้ใช้โปรไฟล์ประเภท Public ได้
        - สามารถค้นหาคอร์สเรียนโดยการค้นหาจาก ชื่อคอร์สเรียน ชื่อเจ้าของคอร์สเรียน/Creator โดยที่คอร์สเรียนนั้น ๆ เปิดสิทธิ์การเข้าถึงผู้ใช้โปรไฟล์ประเภท Public ได้
    - ในแต่ละคอร์สเรียน

        - สามารถเรียกดูตัวอย่าง Video การสอนของคอร์สเรียนนั้น ๆ ได้
        - สามารถเรียกดูรายละเอียดของคอร์สเรียนของคอร์สเรียนนั้น ๆ ได้
        - สามารถเรียกดูข้อมูลการรีวิวคอร์สเรียนของคอร์สเรียนนั้น ๆ ได้
        - สามารถเรียกดู Partner/ผู้สอน ของคอร์สเรียนนั้น ๆ ได้

        - สามารถเข้าดูข้อมูลของเจ้าของคอร์สผ่างทางลัดไปยัง Profile ได้
        - สามารถเข้าดูเนื้อหาในส่วนอื่น ๆ ที่เกี่ยวข้องทางลัดที่มีการสร้างไว้ได้ เช่น เนื้อหาในหมวดหมู่ของเพลงฮิต เนื้อหาในหมวดหมู่คอร์ดเพลง เนื้อหาในหมวดหมู่แบ็คกิ้งแทร็ก
