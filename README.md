## คอร์สเรียนออนไลน์
- สิทธิ์การเข้าถึงคอร์สเรียนเป็น Public, Follower, Member, Private
- Library/คลัง _>> การจัดการโปรไฟล์ > Library/คลัง ของคอร์สเรียนออนไลน์_
    - [ ] สามารถเรียกดูรายการคอร์สเรียนทั้งหมดใน Library/คลัง ได้
    - [ ] สามารถค้นหาคอร์สเรียนโดยการค้นหาจาก ชื่อคอร์สเรียน ชื่อเจ้าของคอร์สเรียน/Creator ได้
    - [ ] สามารถเข้าถึงข้อมูลภายในคอร์สเรียนได้
- ผู้ใช้โปรไฟล์ประเภท Public

        - [ ] สามารถค้นหาคอร์สเรียนโดยการค้นหาจาก ชื่อคอร์สเรียน ชื่อเจ้าของคอร์สเรียน/Creator โดยที่คอร์สเรียนนั้น ๆ เปิดสิทธิ์การเข้าถึงผู้ใช้โปรไฟล์ประเภท Public ได้
